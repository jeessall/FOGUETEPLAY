Jogo Foguet (Pygame)

Um jogo 2D simples feito com Python e Pygame, onde o jogador controla um foguete, atira em inimigos e tenta sobreviver o máximo possível.

Sobre o jogo

- Controle um foguete na parte inferior da tela
- Atire com a tecla ESPAÇO
- Inimigos aparecem no topo da tela
- Destrua os inimigos antes que eles cheguem até você

Tecnologias usadas

- Python 3.11
- Pygame
- Programação orientada a objetos (POO)

Controles

| Tecla | Ação |
|------|------|
| ← / → | Mover o foguete |
| ESPAÇO | Atirar |
| X (fechar janela) | Sair do jogo |

COMO RODAR O PROJETO

Repositorio https://github.com/jeessall/FOGUETEPLAY

Instalar Python 3.11

Baixe em:
https://www.python.org/downloads/

Escolha a opção windows installer 64x

Marque a opção **Add Python to PATH**

-Criar ambiente virtual
No terminal dentro da pasta do projeto:

-bash
py -3.11 -m venv .venv

-Ativar o ambiente virtual
.venv\Scripts\activate

-Instalar dependencias
pip install pygame

-Rodar o game
python main.py
